# rtlink-pro-vue2
本项目是适用于 vue2 框架的前端开源项目，瑞太智联自用的UI组件库。
要求 vue2 版本 >= 2.5.17。
本项目UI部分依赖于 elementUI。

## Project Setup

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Compile and Minify for Production

```sh
npm run build
```

### Lint with [ESLint](https://eslint.org/)

```sh
npm run lint
```
